# Iron and Oak

Iron and Oak is a premium construction and design-build website built with Astro. It presents a fictional construction firm through bold editorial layouts, service pages, and project showcases.

## Highlights

- Home, services, portfolio, about, and contact pages
- Architecture and construction-focused visual direction
- Static Astro site with lightweight asset handling

## Stack

- Astro
- Bun

## Quick start

```bash
bun install
bun run dev
```

Build and preview:

```bash
bun run build
bun run preview
```
